This repository contains my solutions for the TCS Data Visualization Virtual Internship hosted on Forage.
I worked with retail sales data to understand trends, customer behavior, and market opportunities.
I cleaned and explored the dataset to ensure accurate insights for analysis.
Using PowerBI, I created dashboards to answer key questions for the CEO and CMO.
The visuals highlight revenue trends, top customers, high-demand regions, and potential growth markets.
This project helped me improve my skills in data visualization, business analytics, and storytelling with data.
These solutions can serve as a reference for anyone learning PowerBI or data-driven business insights.
